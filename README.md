rfm73
=====

This is a library for the Arduino. Designed originaly to work with the Radioduino.

## Connecting RFM73 to Arduino Uno

| RFM73 pin | signal | Arduino |
| --------- | ------ | ------- |
| 1         | GND    | GND     |
| 2         | VDD    | 3.3V    |
| 3         | CE     | 8       |
| 4         | CSN    | 10      |
| 5         | SCK    | 13      |
| 6         | MOSI   | 11      |
| 7         | MISO   | 12      |
| 8         | IRQ    | 2       |
